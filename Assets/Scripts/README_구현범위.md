# 현재 구현 범위

## 보드

- 3x3 지구 슬롯에서 5개 지구를 가장 밀집된 형태로 선택
- 선형 체인보다 인접 간선 수가 많은 배치 우선
- 지구 크기 6x6
- 지구 사이 간격 3~5칸
- 지구당 2x1 건물 3개
- 지구당 내부 도로 10~16칸
- 인접한 모든 지구를 연결
- 연결당 1~2개 도로
- 지구 간 도로에 굴곡 허용

## AP

- 도로 진입: 1 AP
- 건물 진입: 2 AP
- 일반 탐색: 최소 3 AP
- 라디오 카드 건물 탐색: 1 AP

## 플레이어

- Manager의 Player Count만으로 1~6명 자동 생성
- 프리팹 미지정 시 Capsule 생성
- 기본 스케일 0.2
- 공통 시작 노드 자동 선택
- 같은 칸에 최대 6명
- 인원 변화 시 칸 내부 위치 자동 재배치

## 턴 및 라디오

- 모든 플레이어가 턴 종료 시 라운드 종료
- 기존 라디오 카드 지속시간 감소
- 라디오 카드 3개를 서로 다른 랜덤 건물에 배치
- 지속시간 1~3턴

## 카메라

- WASD
- 방향키
- 우클릭 드래그
- 마우스 휠 줌
- 보드 Bounds 자동 계산
- 생성 직후 자동 포커스
